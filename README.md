# drawing-with-minilibx
A collection of geometric drawings (and not only) built with Escola 42's MiniLibX library, organized into simple and didactic examples for beginners like me

## how to compile

Use the build script:

    $ ./build.sh folder_name/main.c


Requirements:
- The `minilibx-linux` folder must be in the project root.
- You must have X11 development packages installed (Xext, X11, zlib).

Example:

    $ ./build.sh 00_init_window/main.c
